Satzgenerator in `Genie <https://wiki.gnome.org/action/show/Projects/Genie>`_
======================

Kompilieren
-----------

::

	$ valac satzgenerator.gs

Ausführen
---------

::

	$ ./satzgenerator

oder mit Anzahl der Sätze als Parameter::

	$ ./satzgenerator 5
